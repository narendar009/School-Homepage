# 🎓 Bright Future School — Homepage Website

A modern, responsive single-page school website built with **HTML**, **CSS**, and **Bootstrap 5**. This project showcases a professional school homepage with smooth animations, hover effects, and a mobile-first design.

---

## 🌟 Features

- **Sticky Navigation Bar** — Logo, school name, and smooth-scrolling navigation links
- **Hero Banner** — Full-width background image with gradient overlay and CTA button
- **About Section** — Two-column layout with mission & vision cards
- **Facilities Section** — Interactive Bootstrap cards with hover effects
- **Gallery Section** — Responsive image grid with overlay labels
- **Contact Form** — Styled form with validation and a contact-info sidebar
- **Footer** — Quick links, contact details, and copyright info
- **Scroll Animations** — Fade-in-up effects using Intersection Observer

## 🚀 How to Run

1. **Clone or Download** this repository:
   ```bash
   git clone https://github.com/your-username/school-homepage.git
   ```
2. **Open the project folder** and double-click `index.html`, or:
   ```bash
   cd school-homepage
   start index.html        # Windows
   open index.html          # macOS
   xdg-open index.html      # Linux
   ```
3. The page will open in your default browser — **no build step or server required**.

> **Note:** An internet connection is needed on first load to fetch Bootstrap and Google Fonts from CDN.
